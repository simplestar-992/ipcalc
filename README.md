# IPCALC

**IP Calculator & Subnet Helper**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/ipcalc.git
cd ipcalc
go build -o ipcalc .
```

## Usage

```bash
./ipcalc --help
```
