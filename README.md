## Ipcalc

A useful command-line tool for developers.

### Installation

```bash
git clone https://github.com/simplestar-992/ipcalc.git
cd ipcalc
go build -o ipcalc .
```

### Usage

```bash
./ipcalc --help
```

---

## Quick Start

```bash
./ipcalc [command]
```
